SASS+Compass+Susy Grid

This is a demo of a grid. The concept behind it is nesting each row with columns for a total width of 100%.

  • 4: 100%
  • 3: 75%
  • 2: 50%
  • 1: 25%

Nesting .two in .four columns

For demo purposes: this .two has a grey background to show its width. It takes up 3/4 of the total width of its container, a .four, which in turn translates to 100%. Hence, you're nesting "inner" columns in an "outer" column – the container. The sum of all columns on a row should always be 4.
